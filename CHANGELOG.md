# Changelog

All notable changes to this project will be recorded in this document.

## 1.0.5 - 2026-05-28

### Added

- 新增 `atelier / paper / night` 三套界面主题，并支持本地持久化
- 编辑器顶部新增主题下拉切换入口，替代弹窗内的重复主题入口

### Changed

- 主题变量系统继续收口，文字、边框、按钮、表单、帮助中心、调色板和弹窗统一切到 `--theme-*` 语义层
- 中间画布与左侧图片定位预览接入主题同步，不再只切外围壳层
- 工作台工具栏、状态栏、上传区、高级颜色区和弹窗层级做了一轮视觉打磨

### Fixed

- 修正主题切换后图片缩放预览区背景未及时恢复的问题
- 修正顶部主题下拉被右侧高级颜色区遮挡的问题
- 修正中间画布放大后因低精度 backing store 导致的明显发糊问题
- 修正中间画布在高 DPI 屏幕上的网格线清晰度不足问题

## 1.0.4 - 2026-05-27

### Added

- 新增品牌蘑菇图标资源，并接入首页、编辑器顶部和项目信息弹窗
- 新增 `robots.txt`、`sitemap.xml` 和基础结构化数据

### Changed

- 首页标题、站点标题、分享信息和站点描述统一收口到当前品牌文案
- 图片与生成模块的上传区、实时预览开关和说明文案继续压缩，减少首屏占位
- 高级颜色区改为更紧凑的默认两行显示，并补上颜色中文名

### Fixed

- 修正上传图片区文案溢出、实时预览入口过度占位等问题
- 修正近期侧栏样式调整中出现的局部布局撑开问题

## 1.0.3 - 2026-05-27

### Added

- 图片模块增加更直接的实时预览开关，保留原图视图作为默认状态
- 帮助中心补齐模块问号到对应文章的联动入口

### Changed

- 左侧图片区默认显示原图，不再默认持续重算图纸预览
- 项目信息、版本状态与反馈入口继续收敛，减少重复提示和冗余说明
- 裁切空白策略改为更偏向处理边缘浅色留白，贴近当前拼豆图纸使用场景
- 图纸 PNG 导出继续提升清晰度，并保持输出拼豆色号标签

### Fixed

- 避免调整图片预览参数时直接覆盖当前画布内容
- 修正帮助中心打开后目录与正文不同步的问题
- 降低最近一轮图纸生成结果波动，移除默认启用的额外边缘清理干扰
## 1.0.2 - 2026-05-27

### Added

- 中间画布支持双指缩放、双指平移，以及显式“移动画布”工具
- 图片定位预览支持单指拖动、双指缩放
- 编辑器新增“触控帮助”入口，补充平板使用说明
- 编辑器新增统一帮助中心基础版，支持左侧目录、右侧正文和搜索

### Changed

- 页面级 viewport 与编辑器激活态同步阻止浏览器缩放，避免编辑器操作时整页跟着放大缩小
- 工具按钮、颜色块、帮助提示和状态栏提示调整为更适合平板触控的命中区和信息密度
- 图片预览和状态栏文案从桌面鼠标提示改为桌面 / 触控兼容提示
- 字段说明、工具说明、触控说明统一抽离到 `src/features/editor/helpContent.ts`

## 1.0.1 - 2026-05-26

### Added

- README 增加线上地址、界面预览截图、项目 skill 使用提示和更新后的目录结构
- 编辑器顶部品牌标题改为项目信息入口，弹层展示版本号、开源项目地址和反馈邮箱

### Changed

- 项目信息弹层关闭操作改为右上角图标按钮
- 新建确认弹窗底部按钮统一为一致的编辑器紧凑样式

## 1.0.0 - 2026-05-26

### Added

- 单屏首页入口与工作台入口整合
- 三栏编辑器布局与中间独立画布舞台
- 图片上传、缩放、平移、本地预览
- 图片转拼豆图纸能力
- 抖动处理与去背景参数
- 四边坐标尺
- 画笔、橡皮、吸色、填充、框选
- 选区编辑与颜色替换
- 画布整理动作
- 图纸 PNG、颜色清单、工程 JSON 导出

### Changed

- 项目从方案文档阶段收口到首个可发布版本
- 首页、编辑器顶部、侧栏和工具栏进行了多轮压缩与重排
- 本地存储模型调整为单工程缓存
- 图片定位与图纸生成解耦，调整图片参数时不再直接清空当前图纸

### Fixed

- 撤销 / 重做历史栈被持久化流程清空的问题
- 编辑器工作台高度与滚动边界问题
- 画布缩放和平移误参与页面文档流的问题
- 工具栏与侧栏多处重复、冗余、误导性交互
