# Changelog

All notable changes to this project will be recorded in this document.

## 1.0.0 - 2026-05-26

### Added

- 单屏首页入口与工作台入口整合
- 三栏编辑器布局与中间独立画布舞台
- 图片上传、缩放、平移、本地预览
- 图片转拼豆图纸能力
- 抖动处理与去背景参数
- 四边坐标尺
- 画笔、橡皮、吸色、填充、框选
- 选区编辑与颜色替换
- 画布整理动作
- 图纸 PNG、颜色清单、工程 JSON 导出

### Changed

- 项目从方案文档阶段收口到首个可发布版本
- 首页、编辑器顶部、侧栏和工具栏进行了多轮压缩与重排
- 本地存储模型调整为单工程缓存
- 图片定位与图纸生成解耦，调整图片参数时不再直接清空当前图纸

### Fixed

- 撤销 / 重做历史栈被持久化流程清空的问题
- 编辑器工作台高度与滚动边界问题
- 画布缩放和平移误参与页面文档流的问题
- 工具栏与侧栏多处重复、冗余、误导性交互
